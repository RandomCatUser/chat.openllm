## Soft Introduction

chat.openllm is an open-source LLM experience built with Pollination, designed to simulate and explore a cluster-like system directly inside the browser.

Lightweight. Interactive. Customizable.

Perfect for experimenting with AI-style interfaces without heavy backend setup.

## Getting Started

### Clone

```bash
git clone https://github.com/RandomCatUser/chat.openllm
cd chat.openllm
```

### Run

Open `index.html` in your browser.

No installation required.

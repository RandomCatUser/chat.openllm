Soft Introduction

chat.openllm is an open-source LLM experience built with Pollination, designed to simulate and explore a cluster-like system directly inside the browser.

Try Now Lightweight. Interactive. Customizable.

Features

Core

Browser Simulation

Run cluster logic entirely on the client side without backend overhead.

Architecture

Modular Structure

Decoupled files for easier maintenance and community forking.

• Simple navigation system for cluster nodes.

• Minimal CSS and Vanilla JS for easy customization.

• Built to be improved by the community.

Getting Started

Clone the project and explore locally with zero installation required.

1. Clone the repository

git clone https://github.com/RandomCatUser/chat.openllm

2. Navigate and Run

Open index.html in any modern browser.

cd chat.openllm

Privacy & Data Flow

Data Notice

To provide advanced AI simulation features, this application shares your data with Pollinations. Your prompts and interaction data are processed by the Pollinations infrastructure to generate real-time cluster responses.

We do not store your personal data on local servers. Processing happens in real-time.
